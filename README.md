# Ant Design Pro Site

### Development

```bash
$ git clone git@github.com:ant-design/ant-design-pro-site.git
$ cd ant-design-pro-site
$ git submodule update --recursive
$ npm install
$ npm start
```

Then visit http://localhost:8001 .

### Deploy

```
$ npm run site
```

### Publish Components

```
$ npm run publish-components
```
